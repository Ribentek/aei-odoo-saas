# Documentación del proyecto

Esta carpeta reemplaza la GitHub Wiki del repositorio. Las wikis de GitHub para repos
**privados** requieren un plan de pago para poder verse desde el navegador — el contenido
seguía existiendo como un repo git normal (`aei-odoo-saas.wiki.git`), pero nadie podía
leerlo desde la UI. Se migró aquí el 2026-07-10 para que la documentación sea accesible
igual que el resto del código, con PRs y revisión normales.

**Empieza por [Home.md](Home.md)** — es el índice completo (arquitectura, componentes,
operaciones, roadmap, QA).

A partir de ahora, toda la documentación del proyecto se mantiene como archivos `.md` en
esta carpeta — no se vuelve a usar la GitHub Wiki.
